    Relative and Absolute References in Google Sheets

    Relative and Absolute References

    Cells in Google Sheets have unique references, which indicate their location. References are used in formulas to perform calculations, and the fill function can be used to continue formulas sideways, downwards, and upwards.

    Google Sheets has two types of references:

  • Relative references
  • Absolute references

    • Absolute References

    Absolute reference is a choice we make. It is a command that tells Google Sheets to lock a reference. The dollar sign ($) is used to make references absolute.

    Example of relative reference: A1

    Example of absolute reference: $A$1

    References are relative by default, meaning they do not contain a dollar sign ($).

    Absolute References in Google Sheets

    Absolute References

    An absolute reference is when a reference has the dollar sign ($). It locks a reference in the formula, preventing it from changing when using the fill function.

    Add $ to the formula to use absolute references. The dollar sign has three different states:

  • Absolute for column and row: The reference is absolutely locked.
    Example: =$A$1
  • Absolute for the column: The reference is locked to that column, while the row remains relative.
    Example: =$A1
  • Absolute for the row: The reference is locked to that row, while the column remains relative.
    Example: =A$1

    Using Parentheses in Google Sheets

    Parentheses

    Parentheses () are used to change the order of an operation.

    Using parentheses makes Google Sheets calculate the numbers inside the parentheses first, before calculating the rest of the formula.

    Parentheses are added by typing () on both sides of numbers, like (1+2).

    Examples

    No Parentheses

    =10+5*2

    The result is 20 because it calculates 10+10.

    With Parentheses

    =(10+5)*2

    The result is 30 because it calculates (15)*2.

    Formulas with Groups of Parentheses

    =(10+5)+(2*4)+(4/2)

    Note: Cells can be used as values in the formulas inside parentheses, like =(A1+A2)*B5. We have used manual entries in our examples to keep things simple.

    Real Examples in Google Sheets

    Without Parentheses

    The result is 17. The calculation is 2+15, where it uses 15 because 3*5=15.

    With One Parentheses

    The result is 25. The calculation is 5*5, where it uses 5 because it first calculates the numbers inside the parentheses (2+3)=5.

    With Many Parentheses

    The result is 17. The calculation is 5+8+4, where the numbers inside the parentheses are calculated first.

    Nesting Parentheses

    When using more advanced formulas, you may need to nest parentheses. Think of it like an onion with many layers. Google Sheets will calculate the numbers inside the parentheses first, layer by layer, starting with the innermost layer.

    Example Without Nesting

    =2*2+3*4+5*5*2

    It calculates the values flat as you would do with a calculator.

    Example With Nesting

    =((2*2)+(3*4)+(5*5))*2

    Breaking It Down

    Nesting creates layers like an onion. You can have many layers. This example uses two: the inner and outer layers.

    It starts with calculating the numbers in the inner layer:

    =((2*2)+(3*4)+(5*5))*2
          = (4 + 12 + 25)*2
          = (41)*2
          = 82

    Chapter Summary

    Parentheses can be used to change the order of an operation. The numbers inside the parentheses get calculated first. A formula can have multiple sets of parentheses. More advanced formulas can use nesting to create layers of operations, like an onion. Google Sheets calculates the innermost layer first, then the next, and so on.
